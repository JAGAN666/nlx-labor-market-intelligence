Skills × Industries Matrix
Each cell shows what % of jobs in that sector require each skill category. Darker = higher demand. This reveals which industries prioritize communication, technical, clinical, etc. skills.
How to read this
Rows = Industry sectors (classified by O*NET 2-digit code)
Columns = Skill categories (grouped by keyword matching on taxonomy skill names)
Value = % of jobs in that sector that require at least one skill in that category
N Jobs = Number of postings in that sector in the dataset
